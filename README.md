PASOS PARA INSTALAR LOS MODS

1)Descargar el contenido de este repositorio. (El archivo ".md" y ".gitattributes" no se utilizan en la instalación del mod, lo pueden borrar).

2)Abrir la dirrecion en donde esta instalado el juego. (Click derecho en el juego desde la biblioteca de steam ,luego en Administrar->Ver Archivos Locales).

3)Copias las carpetas que previamente descargaste y lo pegas en la carpeta raiz del juego.

4)Listo p wbon, que mas quieres.
